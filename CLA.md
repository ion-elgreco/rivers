## How to Sign

You signify Your acceptance of this Agreement by **signing off every commit** in Your Contribution using Git's `--signoff` (or `-s`) flag:

```bash
git commit -s -m "your message"
```

This appends a trailer to Your commit message of the form:

```
Signed-off-by: Your Name <you@example.com>
```

By adding this trailer, You certify that You have read, understood, and agree to all terms of this Agreement, and that You grant the licenses set forth herein with respect to the Contribution accompanying that commit. The signed-off name and email must be Your real name and a working email address.

Sign-off applies per commit; once You have signed off any commit, You are bound by this Agreement for that commit's content. Pull requests containing any commit without a valid `Signed-off-by` trailer cannot be merged.

If You are contributing on behalf of an employer or other legal entity, You must additionally have authority to bind that entity to this Agreement. If You are not sure whether You have such authority, please contact the Project Owner before signing off any Contribution.
